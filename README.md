## Chapter 05 - গসাগু ও লসাগু (GCD & LCM)
Words:

1. গরিষ্ঠ সাধারণ গুণনীয়ক (গসাগু) = Greatest Common Divisor (GCD)
2. ইউক্লিডের অ্যালগরিদম (a ও b-এর গসাগু = (a - b) ও b-এর গসাগু)
3. লঘিষ্ঠ সাধারণ গুণিতক (লসাগু) = The least common multiple (LCM)
4. Formula: GCD(a, b) x LCM(a, b) = a x b
